## 📋 Over dit project

Dit is mijn persoonlijke portfolio-website, gebouwd met pure HTML en CSS. De site laat zien wie ik ben, wat ik kan en welke projecten ik heb gedaan.

### Inhoud van de site
- **Over mij** — Wie ik ben en wat mijn doelen zijn
- **Skills** — Overzicht van mijn technische vaardigheden
- **Projecten** — Projecten die ik heb gebouwd (home lab, CTF, cloud, netwerk)
- **Certificeringen** — Behaalde en lopende certificaten
- **Contact** — Hoe je mij kunt bereiken

---

## 🛠️ Gebouwd met
